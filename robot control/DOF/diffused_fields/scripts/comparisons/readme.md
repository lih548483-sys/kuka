# Diffused Fields - Baseline Comparison & Ablation Studies
# 几何扩散场 - 基线对比与消融实验

This subdirectory contains validation and evaluation scripts designed to benchmarks the proposed **DOF (Diffusion Orientation Fields)** method against traditional geometric baseline methods and alternative diffusion pipelines.

本目录包含用于验证和评估的实验脚本，旨在将本项目提出的 **DOF（扩散方向场）** 方法与传统几何基线方法以及其他备选扩散流水线进行对比与基准测试。

---

## Scripts Architecture & Comparison Dimensions / 脚本架构与对比维度

### 1. Nearest Frame Comparison (`nearest_frame_comparison.py`)
* **English**: Compares the continuous DOF field against a 0-order interpolation baseline (**Nearest Frame**). For any point in the volumetric grid, the baseline strictly assigns the local frame of its closest vertex on the manifold. This script highlights the geometric discontinuities and singularities generated by naive extension versus the smooth, harmonic gradients produced by the Walk-on-Spheres (WoS) orientation diffusion.
* **中文**: 将连续的 DOF 场与零阶插值基线（**最邻近坐标系**）进行对比。对于体网格中的任意点，基线方法直接赋予其流形表面最近顶点的局部坐标系。该脚本突出了朴素延展带来的几何不连续性与奇异点，并证明了通过球面走样法（WoS）进行方向扩散所带来的平滑谐波梯度。

### 2. Independent Vector Diffusion & Orthonormalization (`vector_diffusion_comparison.py`)
* **English**: A rigorous ablation study evaluating direct $SO(3)$ rotation diffusion against independent Euclidean vector diffusion paired with post-hoc orthonormalization:
  * **Method 1 (Z-Fixed)**: Diffuses vectors independently, keeps the normal ($z$-axis) fixed, and reconstructs the tangent plane via Gram-Schmidt/cross products.
  * **Method 2 (SVD-based)**: Performs Singular Value Decomposition ($A = U \Sigma V^T$) on the diffused non-orthogonal matrix to find the closest orthogonal matrix $R = UV^T \in SO(3)$.
  The script quantitatively demonstrates that independent diffusion introduces geometric shear distortions, whereas explicit orientation diffusion preserves the intrinsic manifold metrics.
* **中文**: 这是一个严格的消融实验，评估了直接在 $SO(3)$ 旋转流形上进行方向扩散，与在欧氏空间中独立进行向量扩散并配合后处理正交化的性能差异：
  * **方法 1（Z轴固定）**：独立扩散各个向量，保持法方向（$z$轴）固定，通过施密特正交化/叉乘强行重构切平面。
  * **方法 2（基于SVD）**：对扩散后的非正交矩阵进行奇异值分解（$A = U \Sigma V^T$），求解最接近的旋转矩阵 $R = UV^T \in SO(3)$。
  该脚本定量地证明了独立向量扩散会引入几何剪切失真，而显式方向扩散能完美保持流形的内在度量。

### 3. Vector Projection Baseline (`vector_projection_comparison.py`)
* **English**: Quantifies the angular deviation on the manifold surface between intrinsic local frames established via scalar diffusion fields and extrinsic frames generated via standard vector projection. 
* **中文**: 定量分析并评估在流形表面上，通过本征标量扩散场建立的局部坐标系与通过标准外部向量投影生成的坐标系之间的角度偏差（Angular Deviation）。

## Verification Pipeline & Visualization / 验证流程与可视化

All scripts leverage **Polyscope (`ps`)** as the visualization backend. For rigid statistical validation, the scripts include built-in deviation analysis routines.

所有脚本均采用 **Polyscope (`ps`)** 作为可视化后端。为了进行严格的统计学验证，脚本内置了偏差分析程序。


### Key Visualization Features / 核心可视化功能:
1. **Angular Deviation Analysis (`visualize_angular_deviations`)**: Computes the exact dot-product angular errors between different orientation fields and projects the error metrics as heatmaps directly onto the mesh/grid vertices.
2. **Cross-Sectional Slicing (`extract_plane`)**: Slices the 3D bounding box volumetric grid along specific geometric planes (e.g., $x=center$) to visually inspect internal volumetric field distribution.

### 核心可视化特性:
1. **角度偏差分析 (`visualize_angular_deviations`)**: 计算不同方向场之间的点积角度误差，并将误差指标作为热力学图直接投影到网格/顶点上。
2. **截面切片 (`extract_plane`)**: 沿着特定的几何平面（如 $x=中心点$）对 3D 包围盒体网格进行切片，以直观检查内部体积场的分布。

## Execution / 运行方式

Execute the comparison benchmarks via standard Python interface:
请通过标准 Python 接口运行对比基准测试：

```bash
# Evaluate against nearest-frame interpolation
python nearest_frame_comparison.py

# Evaluate against independent vector diffusion + SVD/Gram-Schmidt
python vector_diffusion_comparison.py

# Evaluate manifold surface vector projection deviations
python vector_projection_comparison.py
